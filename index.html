2. 前沿洞察

从行业叙事看，本周最重要的两条线索分别是世界模型基础设施化，以及机器人集成商向数据与模型驱动的 Neo-Integrator 升级。

Deedy Das / X

世界模型正在成为机器人规模化的关键基础设施

过去 18 个月世界模型赛道吸引约 100 亿美元投入，讨论从“生成视频”扩展到为机器人基础模型提供可扩展训练数据与物理直觉。核心问题不再只是画面逼真，而是能否同时具备长期一致性、交互可控性、物理因果、空间记忆与可用于机器人训练的反馈信号。

  • 代表项目覆盖 Google Genie、Tesla Optimus、NVIDIA DreamDojo / DreamZero、Microsoft Muse、World Labs、Runway、Skild、Physical Intelligence、Figure 等。
  • 真正机会在于把世界模型从内容生成工具变成机器人训练、评估、预演和部署校准的基础设施。
Avi Zurlo / X

Neo-Integrator 将成为机器人落地价值捕获层

传统工业机器人部署中，硬件通常只占总成本的 40-50%，其余主要来自系统集成、开发、调试、安全与运维。Robot Learning 会压缩开发成本，但现场集成仍高度垂直、劳动密集，因此“懂模型 + 懂垂直场景 + 拿得到客户现场数据”的 Neo-Integrator 有望捕获部署剩余价值。

  • 打法演进：Teleoperation 先创收并积累训练数据；半自主化提升毛利；最终向 Zero-shot Deployment 过渡。
  • 适合创业切入的场景通常具备弱劳动经济学、任务重复度高、现场集成复杂、客户愿意为可靠性和托管服务付费等特征。

5. 技术进展

18 项技术进展可以归为世界模型、VLA/机器人基础模型、实时推理、长程记忆、控制与安全鲁棒几条主线。

World Model / VLA

RAW-Dream：任务无关世界模型里的零样本“梦境训练”

微软亚洲研究院、南京大学、UIUC、武汉大学、悉尼科技大学、清华大学

核心看点：把世界模型的通用物理先验与下游任务数据解耦，让 VLA 在“想象沙盒”里后训练。

关键结果：跨模拟与真实环境取得持续增益，目标是降低现实交互数据成本。

AI for Science

领域自适应 ML + XPCS 量化非平衡晶界动力学

麻省理工学院量子测量研究组、MIT 计算科学与工程中心

核心看点：用理论模拟训练 AI，再迁移到高噪声 X 射线实验数据，提取材料微观动力学参数。

关键结果：首次从复杂波动信号中解析晶界浓度、扩散率、刚度等指标。

Safety / VLA

DreamAvoid：关键阶段测试时梦境预演，降低 VLA 失败率

香港大学、香港科技大学、西北大学

核心看点：在精细操作关键期生成候选动作并短视距推演，提前识别会导致失败的动作。

关键结果：在真实与模拟机械臂任务中减少灾难性动作，提升整体成功率。

RL / Control

DR.Q：去偏置表示提升样本高效连续控制

腾讯混元、麦吉尔大学、北京大学计算机学院

核心看点：通过互信息最大化和衰减优先级回放，减少早期经验与状态表征偏差。

关键结果：一套超参数在多项连续控制基准中匹配或超过强基线。

Interface / Agent

Engagement Process：重构观察与行动的时间接口

小鹏机器人

核心看点：把观察和行动从固定回合制解耦为异步事件流，处理延迟、长周期动作与多频协作。

关键结果：揭示传统 step 接口忽略的时序问题，让策略适应显式时间成本。

VLA / Policy

AttenA+：用逆速度场重加权关键动作

香港科技大学（广州）、香港大学、中国科学技术大学、IDEA、南方科技大学、X-Humanoid

核心看点：不改模型结构，只通过速度驱动注意力让模型更关注低速高精度关键片段。

关键结果：在 LIBERO 等基准中将 OpenVLA-OFT 成功率提升至 98.6%，并在真实 Franka 测试中验证泛化。

Bimanual / Policy

CUBic：统一双臂感知与控制

北京航空航天大学、西湖大学、浙江大学、北京大学

核心看点：用双向感知协调字典与统一扩散策略，让左右臂在共享表征中自然协同。

关键结果：在 RoboTwin 双臂任务中提升协调准确性与成功率。

Realtime VLA

Realtime-VLA FLASH：投机推理压缩 dVLA 延迟

中科院计算所、中国科学院大学、南京大学、Dexmal

核心看点：以轻量草稿模型快速生成动作，再由主模型并行验证，并在高难动作相位回退。

关键结果：平均推理延迟降至 19.1ms，约 3.04 倍提速，支撑 50Hz 高频闭环控制。

Navigation

HCSG：动态人群中的语义-几何导航

香港科技大学（广州）、清华大学（深圳）、北京科技大学、NUS、上海交大

核心看点：同时预测人类轨迹与理解人类行为语义，把“躲人”升级为“看懂人”。

关键结果：HA-VLNCE 中成功率提升 14%，碰撞率下降 34%。

Robust VLA

PAIR-VLA：视觉突变下该忽略什么、该反应什么

香港科技大学、微软亚洲研究院、浙江大学

核心看点：通过成对视觉变体和不变性/敏感性辅助目标，让 VLA 忽略无关视觉变化、响应任务相关变化。

关键结果：在 ManiSkill3 严苛测试中微调性能最高提升 16.62%。

World Model

SANA-WM：2.6B 参数实现分钟级 720p 世界模型

英伟达

核心看点：Hybrid Linear Attention、双分支相机控制与 long-video refiner 支撑一分钟相机可控视频生成。

关键结果：使用约 213K clips 训练；distilled variant 可在单张 RTX 5090 上运行，降低长视频世界模型门槛。

Unified Embodied Model

Pelican-Unified 1.0：统一理解、推理、想象与行动

北京人形机器人创新中心（X-Humanoid）

核心看点：用同一 VLM 与未来生成器闭环连接视觉理解、语言推理、未来视频和动作生成。

关键结果：8 个 VLM 基准平均 64.7；WorldArena EWM Score 66.03；RoboTwin 平均成功率 93.5%。

Benchmark / Agent

LongAct & HoloMind：长程家务基准与记忆型 Agent

中科院自动化所、中关村学院、中国科学院大学

核心看点：把家务任务建成数千步长程执行问题，并用 DAG 规划、空间记忆、情景记忆和 Critic 形成闭环。

关键结果：GPT-5 在 HoloMind 下目标完成率 59.0%、全任务成功率约 16%，显示长程执行仍是瓶颈。

Scene Understanding

SceneParser：连接物体、部件与可操作 affordance

哈尔滨工业大学（深圳）、中关村学院、华中科技大学、北邮、华南理工、牛津、北大

核心看点：把场景解析为 scene-object-part-affordance 层级链，让视觉理解直接服务交互规划。

关键结果：构建含 110K 训练图像、1.74M affordance 标注的数据集，并在层级解析上优于拼接式管线。

Long-horizon Policy

DSSP：全历史状态空间策略提升长程操作

上海交通大学

核心看点：用 Mamba 类 SSM 把完整多模态历史压缩为 context token，再条件化扩散动作生成。

关键结果：RoboTwin 2.0 平均相对提升 12.8%；真实机器人平均成功率从 30% 提升到 70%。

Humanoid Motion

HoloMotion-1：视频衍生动作语料驱动人形全身跟踪

地平线机器人

核心看点：用 2000 小时以上视频重建动作、动捕和内部数据训练稀疏 MoE Transformer，并结合 KV-cache 与序列级 PPO。

关键结果：未见动作评测成功率 97.55%，关键点误差较强基线降低约 40%，真实平台可 200-300Hz 运行。

Diffusion Decision

Ada-Diffuser：潜变量识别增强扩散式长程决策

UCSD、CMU、MBZUAI、斯坦福、约翰斯·霍普金斯大学

核心看点：从短时间块中识别看不见的动态因素，并把潜变量估计嵌入扩散规划和策略学习。

关键结果：覆盖 8 个环境 23 种设置，在 MuJoCo、Maze2D、RobotMimic、LIBERO-10 等任务中优于多类扩散基线。

World Model / Feedback

Feedback World Model：用实时反馈校准扩散策略

南洋理工大学 MARS 实验室

核心看点：推理时用动作后的真实观察更新轻量反馈状态，让世界模型持续自我校准。

关键结果：最高减少 76.4% 世界模型预测误差，分布外成功率提升 30%。

6. 创业机会地图

基于本周内容补充的机会方向：更适合早期团队从工具链、数据、部署和垂直场景切入，而不是直接与大模型实验室正面竞争。

01

世界模型基础设施

SANA-WM、RAW-Dream、Feedback WM 都指向同一趋势：世界模型需要相机轨迹、物理尺度、反馈校准和任务可控。

早期切入：做高质量 pose/3D 标注流水线、机器人 rollout 评测、世界模型数据清洗与合成、在线反馈校准中间件。

02

垂直 Neo-Integrator

模型能力会逐步通用，但客户现场、设备接入、安全合规和运维仍高度垂直。

早期切入：从 3PL 仓储、质检、餐饮后厨、制造搬运、危险工种等弱劳动经济学场景切入，先用遥操作托管服务获取收入与数据。

03

机器人数据工厂

Data Factory、EgoTwin、视频衍生动作语料说明，数据质量和采集成本仍是瓶颈。

早期切入：建设遥操作采集、第一视角手部/物体 3D 对齐、自动纠错、仿真增强和任务标签体系。

04

实时 VLA 与边缘部署

FLASH、HoloMotion-1 都表明大模型必须在受限算力上实时闭环。

早期切入：做投机推理、稀疏激活、KV-cache、动作验证、嵌入式运行时和机器人端模型压缩工具。

05

鲁棒与安全中间件

DreamAvoid、PAIR-VLA、HCSG 都在解决“会做”到“稳定安全地做”的鸿沟。

早期切入：做关键阶段避险、视觉扰动鲁棒训练、社交导航、安全边界评估和失败样本管理。

06

长程记忆与任务规划

LongAct、DSSP、SceneParser 指向从单步技能到长程任务的系统升级。

早期切入：做家庭/工业长程任务基准、空间/情景记忆、层级场景解析、可操作 affordance 和过程级 Critic。

07

教育与开发者平台

Classover 与 Stretch 4 显示市场需要从硬件、课程到开发工具的低门槛入口。

早期切入：面向 K-12、工程师和企业客户提供具身 AI 课程、标准化机器人平台、开发者 SDK 与任务模板。
